## Install Pygments on OS X

* Install latest python (see [python docs](http://docs.python-guide.org/en/latest/starting/install/osx/)):
  * ```$ brew install python ```

* Install pygments (see [pygments.org](http://pygments.org/)):
  * ``` $ pip install pygments ```
